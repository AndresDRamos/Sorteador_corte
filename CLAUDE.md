# Proyecto: Sorteador de corte

## Contexto

Aplicación web que lee archivos DXF de nesteos (layouts de corte láser), 
los parsea, y los renderiza en una aplicación web, y extraer los bloques de números de parte que conformen el nesteo para enlistarlos y agruparlos.

## Stack técnico

- Backend: Node.js con Express

- Nombres de funciones en inglés
